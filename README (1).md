> **פלטפורמת ניהול פיננסי אישי מתקדמת בעברית עם יועץ AI חכם** 🤖

FlowEco עוזרת לישראלים לנהל את הכספים בצורה חכמה ופשוטה, עם תמיכה מלאה בכרטיסי אשראי, מנויים ותקציבים.

---

## ✨ תכונות עיקריות

🏠 **Dashboard חכם** - סיכום פיננסי עם גרפים ותובנות  
💸 **ניהול הוצאות** - מעקב מדויק עם קטגוריות מותאמות  
💰 **ניהול הכנסות** - הכנסות קבועות ומשתנות  
📊 **תקציבים** - התראות אוטומטיות בחריגה מתקציב  
🎯 **יעדי חיסכון** - מעקב התקדמות ויזואלי  
🔔 **ניהול מנויים** - התראות על חידושים קרבים  
🤖 **יועץ AI** - המלצות אישיות בעברית  
📱 **PWA** - חוויה כמו אפליקציה במובייל  

---

## 🚀 טכנולוגיות

**Frontend:**
- WordPress + Elementor Pro
- JavaScript ES6+ (WPCodeBox)
- CSS Grid & Flexbox
- PWA Support

**Backend:**
- Cloudflare Workers (Serverless)
- Cloudflare D1 Database (SQLite)
- Resend Email API
- OpenAI GPT-4o-mini

**Infrastructure:**
- Cloudflare CDN
- Custom JWT Authentication
- RESTful API Design
